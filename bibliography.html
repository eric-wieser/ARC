stackoverflow.com

Stackoverflow is a community-driven Q&A site for programmers. If you ask a question there, you typically get a bunch of responses within 15 minutes. It's an incredibly useful resource for learning to program, and for teaching others. The following are questions asked by me.

  • What is causing this to behave strangely?
  • How can I extract out sensitive information when pushing to a git repository?
  • Alternative to System.DateTime.Now.Ticks in the .NET Micro Framework?
  • Is there a better way of writing this code?
  • Should I use a property or a method for a calculated value? [closed]
  • What's the difference between I2CDevice.CreateReadTransaction() and new I2CDevice.I2CReadTransaction()?
  • How can I prevent invalid configuration of a Microsoft.SPOT.Hardware.InterruptPort?

tinyclr.com/forum

This is a community run by GHI Electronics, the designers of the FEZ Panda II. The forums were very useful for getting information about the workings of the controller. The following are the questions I asked on the forum.

  • FEZ Project Templates not showing up in Visual Studio 2010 Professional SP1
  • .NET Micro Math library - or lack of it
  • System.Exception - CLR_E_DIVIDE_BY_ZERO occurs when dividing two doubles
  • PinCapture-ing on multiple pins simultaneously
  • Using interrupts to measure the duty cycle of a PWM signal
  • Setting interrupts on a TriStatePort
  • Structs vs Classes
  • Debugger disagrees with the FEZ Panda II!
